# 桌面倒计时

基于 Python + tkinter 的简洁桌面倒计时软件。

## 功能

- 手动设置倒计时时长（时/分/秒）
- 快捷预设时长（5 分 / 10 分 / 15 分 / 25 分 / 30 分 / 45 分 / 1 时 / 2 时）
- 开始 / 暂停 / 重置（重置时自动清空输入）
- 实时进度条，直观显示剩余比例
- 最后 10 秒数字红色闪烁提醒
- 窗口透明度可调（滑块调节 0.2 ~ 1.0）
- 倒计时结束系统提示音
- 显示预计结束时间（北京时间，精确到秒）
- 窗口置顶

## 运行方式

```bash
python countdown.py
```

或双击 `启动倒计时.bat`

## 依赖

- Python 3（自带 tkinter）
- Windows 系统（使用 winsound 发声）
