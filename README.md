# 桌面倒计时

基于 Python + tkinter 的简洁桌面倒计时软件。

## 功能

- 手动设置倒计时时长（时/分/秒）
- 开始 / 暂停 / 重置
- 倒计时结束系统提示音
- 显示预计结束时间（北京时间，精确到秒）
- 窗口置顶

## 运行方式

```bash
python countdown.py
```

或双击 `启动倒计时.bat`

## 依赖

- Python 3（自带 tkinter）
- Windows 系统（使用 winsound 发声）
